# Day 9

## Completed

* Removed Java's built-in sorting approach using `Comparator`.
* Implemented manual Bubble Sort to sort students by ID.
* Learned how to swap objects inside an `ArrayList`.
* Tested sorting with multiple student records.
* Understood the importance of algorithm implementation instead of relying on library methods.

## Learned

* Bubble Sort algorithm
* Nested loops
* Swapping objects using a temporary variable
* Time Complexity - O(n²)
* Space Complexity - O(1)
* Stable sorting
* Best, Average and Worst case of Bubble Sort

## Reflection

* Today was my first time implementing a sorting algorithm manually.
* It helped me understand what Java's `Collections.sort()` is doing internally.
* Swapping objects and nested loops were confusing initially, but became much clearer after implementation.
* Biggest takeaway: Knowing how an algorithm works is more valuable than simply using a built-in method.
